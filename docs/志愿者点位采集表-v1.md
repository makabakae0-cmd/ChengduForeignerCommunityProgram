# 志愿者点位采集表 v1

## 适用范围

本表用于志愿者收集社区地点、地标、商户、公共设施等 `places` 信息。

目标不是让志愿者直接填写后台完整地点数据，而是形成一张可审核、可复核、可转录入库的点位采集卡。

## 填写原则

- 不确定就写“不确定”，不要猜。
- 地址优先复制地图平台的标准地址，推荐高德地图或腾讯地图。
- 每条记录至少提供一个可复核的位置凭证。
- 英文名不知道可以留“待补充”，不要现场硬翻译。
- 遇到多入口、商场内、园区内、背街门店等情况，必须补充入口说明。
- 没有实地到访时，默认勾选“需要复核”。

## 字段清单

### A. 基本识别

| 字段 | 级别 | 填写说明 |
| --- | --- | --- |
| 中文名 | 必填 | 以门头、招牌、官方名称、地图名称中最稳定的中文称呼为准。 |
| 英文名/英文别称 | 建议填写 | 有官方英文名就填官方英文名；没有就写“待补充”。 |
| 点位类型 | 必填 | 从上方类型表中选择最接近的一项。 |
| 别名/旧名/常用叫法 | 选填 | 如居民常用简称、旧店名、地图别名。 |

### B. 位置

| 字段 | 级别 | 填写说明 |
| --- | --- | --- |
| 详细地址 | 必填 | 优先复制高德/腾讯地图标准地址，不要只写“某地铁口附近”。 |
| 位置凭证 | 必填 | 至少提供 1 项：定位分享、地图截图、POI 页面截图、地图链接。 |
| 入口说明 | 条件必填 | 商场内、多入口、园区内、背街、地图经常导错时必须填写。 |
| 路线补充备注 | 选填 | 如“从 A 口出站右转 200 米”“需从 2 楼进入”。 |

### C. 使用信息

| 字段 | 级别 | 填写说明 |
| --- | --- | --- |
| 主要服务/这是做什么的 | 必填 | 用 1 到 3 句话说明这个点位主要提供什么服务。 |
| 营业/开放时间 | 建议填写 | 现场门牌、官方页面、店员口述均可；不确定请标注。 |
| 联系方式 | 建议填写 | 电话、微信、公众号、小程序、预约方式任选其一。 |
| 适合人群 | 建议填写 | 如新居民、亲子家庭、外籍居民、老年人、上班族。 |
| 费用信息 | 条件必填 | 收费类点位尽量填写，如免费、低消费、人均区间、课程价格段。 |
| 语言支持 | 建议填写 | 如中文、英文、双语菜单、可简单英文沟通。 |
| 无障碍情况 | 选填 | 如轮椅可进、电梯、无障碍卫生间、坡道。 |
| 标签 | 选填 | 尽量从固定词里选，如英语友好、亲子、无障碍、24 小时、可停车、支持国际卡、需预约。 |
| 现场经营/开放状态 | 建议填写 | 如正常营业、暂停营业、已关闭、不确定。避免与后台“发布状态”混淆。 |

### D. 可信度

| 字段 | 级别 | 填写说明 |
| --- | --- | --- |
| 信息来源 | 必填 | 例如实地观察、店员口述、官方公众号、地图平台、居民推荐。 |
| 采集时间 | 必填 | 写明日期；如可能，补充具体时间段。 |
| 采集人 | 必填 | 写姓名、昵称或志愿者编号。 |
| 是否实地到访 | 必填 | 只能选“是/否”。 |
| 是否需要复核 | 必填 | 只能选“是/否”。 |
| 复核原因 | 条件必填 | 标记“需要复核”时必填，例如未实地到访、时间不确定、地址与招牌不一致。 |

### E. 媒体

| 字段 | 级别 | 填写说明 |
| --- | --- | --- |
| 门头/入口照片 | 条件必填 | 正常情况下至少 1 张；没有明显门头时可用入口或明显识别物代替。 |
| 环境照片 | 建议填写 | 建议补充周边环境、楼层入口、内部环境图。 |
| 补充备注 | 选填 | 如“地图会导到背门”“晚上招牌不明显”“工作日和周末时间不同”。 |

## 推荐填写方式

### 最小可用版

如果时间有限，至少保证以下字段完整：

- 中文名
- 点位类型
- 详细地址
- 位置凭证
- 主要服务
- 信息来源
- 采集时间
- 采集人

### 完整推荐版

若条件允许，建议补齐：

- 英文名/英文别称
- 入口说明
- 营业/开放时间
- 联系方式
- 适合人群
- 费用信息
- 语言支持
- 无障碍情况
- 门头/入口照片
- 环境照片

## 志愿者常见问题

### 1. 地址从哪里复制？

优先从高德地图或腾讯地图复制标准地址。二者不一致时，优先保留你现场观察到的门头信息，并在备注里写明。

### 2. 一定要给经纬度吗？

不用。志愿者阶段更重要的是提供可复核的位置凭证。经纬度和最终地图点位由后台审核时统一处理。

### 3. 英文名没有怎么办？

可以写“待补充”。如果只是你个人翻译，除非非常确定，否则不要当作正式英文名提交。

### 4. 哪些情况一定要写入口说明？

以下情况视为必须填写：

- 点位在商场、写字楼、园区、学校、医院内部
- 同一地址有多个入口
- 地图导航经常导错
- 门头与地图名称明显不一致

### 5. 哪些情况一定要标记“需要复核”？

建议至少包括以下情况：

- 你没有实地到访
- 营业时间、联系方式、名称不确定
- 地图地址和现场招牌冲突
- 点位近期可能搬迁、装修或暂停营业

## 建议提交格式

每条点位建议至少包含：

- 文字信息 1 份
- 位置凭证 1 份
- 图片 1 到 3 张

如果用表格收集，可将“位置凭证”和“图片”填为链接或文件名。
